Getting Started

Quick Start Guide

Welcome to lvlBase! This guide will help you get your organization up and running in under 10 minutes.

Step 1: Create your account

Sign up at lvlbase.io/register and choose your organization type. Your account comes with a 14-day free trial of Pro features.

Step 2: Configure your workspace

After signing in, the Setup Wizard will guide you through uploading your org structure, inviting initial users, and configuring your first modules.

Step 3: Install the SDK (optional)

For custom integrations, install our JavaScript SDK:

// Install via npm npm install @lvlbase/sdk // Or via CDN <script src="https://cdn.lvlbase.io/sdk/v2.js"></script>

Step 4: Initialize the SDK

import { LvlBase } from '@lvlbase/sdk'; const client = new LvlBase({ apiKey: 'YOUR_API_KEY', orgId: 'YOUR_ORG_ID', environment: 'production' }); // Test the connection const status = await client.ping(); console.log(status); // { status: 'ok', latency: 24 }
Authentication

Authentication Overview

lvlBase supports multiple authentication methods to fit your organization's security requirements.

Supported Methods

Email/Password
Standard email and password authentication with MFA support.
Google OAuth
Single-click login with Google accounts via OAuth 2.0.
SAML 2.0 SSO
Enterprise SSO with Okta, Azure AD, OneLogin, and more.
API Keys
Secure API keys for service-to-service authentication.

API Authentication Example

// All API requests require an Authorization header fetch('https://api.lvlbase.io/v2/users', { headers: { 'Authorization': `Bearer YOUR_API_KEY`, 'Content-Type': 'application/json', 'X-Org-ID': 'YOUR_ORG_ID' } }) .then(res => res.json()) .then(data => console.log(data));
API Reference

Users API

The Users API allows you to manage organization members programmatically.

List Users

GET /v2/users // Response { "users": [ { "id": "usr_abc123", "name": "Jane Smith", "email": "jane@org.com", "role": "manager", "skillDNA": { "level": 7, "topSkills": ["leadership", "data-analysis"] }, "guild": "Alpha Squad" } ], "total": 2847, "page": 1 }

Create User

POST /v2/users { "name": "John Doe", "email": "john@org.com", "role": "employee", "department": "Engineering", "sendInvite": true }

Webhooks

lvlBase can send real-time webhook notifications to your endpoints when key events occur in your organization.

Available Events

user.created // New user joined user.skill_updated // Skill DNA profile changed guild.level_up // Guild reached new level workflow.triggered // Automation workflow fired report.generated // Scheduled report ready ai.insight_ready // New AI insight available

Webhook Payload Example

{ "event": "guild.level_up", "timestamp": "2025-01-15T14:32:00Z", "orgId": "org_xyz789", "data": { "guildId": "guild_alpha", "guildName": "Alpha Squad", "newLevel": 5, "totalXP": 12400 } }
